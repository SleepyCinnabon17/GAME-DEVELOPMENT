## 🕹 **About This Repository**

This repository contains my **independent game development projects**, created primarily using **Godot and Unity**.

I design and build games from the ground up — from **game mechanics and scripting to art assets and UI design**.  
All visual elements in these projects are **hand-drawn or custom made**, emphasizing unique style and creative experimentation.

The goal of these projects is to explore **game systems, player interaction, and expressive visual design** while strengthening technical skills in game programming.

---

## ⚙️ **Development Focus**

- 🎮 Gameplay systems and mechanics  
- 🧠 Game logic and scripting  
- 🎨 Hand-drawn assets and visual design  
- 🖥 UI and interaction systems  
- 🧩 Level and world design  

---

## 🧠 **Game Architecture**

Typical components across projects include:

| Component | Purpose |
|-----------|--------|
| GameManager | Core gameplay flow and state control |
| PlayerController | Player movement and interaction |
| EnemyAI | Enemy behavior and logic |
| UIManager | Interface elements and HUD |
| LevelSystem | Scene and level management |
| AssetPipeline | Custom sprites and visual assets |
